www.dineshonjava.com : Web Microservices Application

Overview

  • This application defines a simple web-application for accessing student/employee data.
  • All the student/employee information is fetched via a RESTful interface to a students/employees microservice.

The Student Service

  • Eureka Dashboard: http://localhost:1111
  • Check applications registered: http://localhost:1111/eureka/apps/
  • Fetch student #1: /students/1
  • Fetch by email: /students/1
  • Student Search

The Employee Service

  • Eureka Dashboard: http://localhost:1111
  • Check applications registered: http://localhost:1111/eureka/apps/
  • Fetch Employee #1: /employees/1
  • Fetch by email: /employees/1
  • Employee Search
